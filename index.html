¿Qué es HTML?

HTML, o HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar para la creación de páginas web y aplicaciones web. Es utilizado para crear la estructura y el contenido de una página web al definir y organizar el contenido en forma de encabezados, párrafos, listas, enlaces, imágenes y otros elementos.
